#  Roni Dey - Portfolio

Welcome to my personal portfolio!  
This is a high-performance, visually stunning digital experience built to showcase my journey as a Full-Stack Developer, my projects, and my technical expertise.

<img width="1919" height="876" alt="image" src="https://github.com/user-attachments/assets/b23551d8-6dcc-4573-bec7-584710b0da58" />

##  Features

- **Modern UI/UX** – A sleek, dark-themed aesthetic with a premium feel.
- **Dynamic Animations** – Powered by **GSAP** and custom Tailwind animations for smooth transitions and hover effects.
- **Smooth Scrolling** – Integrated **Lenis** for a silky-smooth browsing experience.
- **Interactive Navigation** – Numbered menu system with animated indicators and section-aware tracking.
- **Project Showcases** – Distinct project cards featuring browser-frame visuals, tech stack badges, and integrated call-to-actions.
- **Awards Timeline** – Custom-designed vertical timeline with Lucide Trophy icons and glowing hover states.
- **Responsive Design** – Fully optimized for mobile, tablet, and desktop.
- **Contact System** – Functional contact form with integrated Email, GitHub, and LinkedIn links.

##  Tech Stack

- **Framework:** Next.js 15 (App Router)  
- **Language:** TypeScript  
- **Styling:** Tailwind CSS  
- **Animations:** GSAP  
- **Icons:** Lucide React  
- **Smooth Scroll:** Lenis  
- **Deployment:** Vercel  

##  Getting Started

### 1️ Clone the Repository

```bash
git clone https://github.com/RoniDey1819/Portfolio.git
cd Portfolio
```

### 2️ Install Dependencies

```bash
pnpm install
# or
npm install
```

### 3️ Start Development Server

```bash
pnpm dev
# or
npm run dev
```

### 4️ Open in Browser

Visit:  
http://localhost:3000


##  Project Structure

```
/app        → Main routing and page logic  
/components → Reusable UI components  
/lib        → Utility functions & global data  
/public     → Static assets (images, logos, thumbnails)  
```
